# DiplomadoiOS_5_AppCalculadoraIMC
App numero 5 del diplomadoiOS en MobileStudio, aplicando el patron MVC, es una calculadora de indice de masa corporal que cambia de pantalla y dependiendo del resultado muestra algo diferente en cada pantalla
